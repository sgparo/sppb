# ProRoof Estimator

Colorado Front Range Edition - Roofing Calculator App

A React-based roofing estimate calculator with support for solar panel removal & replacement pricing, pitch adjustments, and waste factor calculations.

## Setup

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:5173`

## Features

- Adjustable roof area (sq ft) with visual square conversion
- Roof pitch settings with category classification
- Dynamic pitch surcharges (low/moderate/steep)
- Solar panel removal & replacement pricing
- Electrical upgrade cost estimation
- Waste factor adjustments
- Real-time cost calculations
- Print quote functionality

## Tech Stack

- React 18
- Tailwind CSS
- Lucide React Icons
- Vite
